OpenPension-client
==================

OpenPension Client is the Web interface for the OpenPension Project  
aimed at revealing the secrets behind the pension market.


Our stack
=========
[NodeJS] [1]  
[ExpressJS] [2]  
[Jade] [3]  
[Squel] [4]  
[PostgreSQL] [5]  

Also used
===============
[Templatizer] [6]  
[Tabletop] [7]  

Getting Started
===============
1.  Clone the repo

        >git clone ...
        >git checkout marionette
2.  Update submodules               
        
        >git submodule init 
        >git submodule update
3.  Install [NodeJS] [1]
4.  Install packages
        
        >npm install
        >npm install bower -g
        >bower install
        >npm install grunt -g
        >grunt

5.  Install node-dev (globally)

        >npm install node-dev -g
6.  Run the server 

        >node-dev app.js
